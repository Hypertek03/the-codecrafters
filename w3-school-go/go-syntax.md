Go syntax are different files in Go which makes up the program structure.

There consist of different parts:

1. Package declaration: in every program in GO there is a package, so therefore it is defined or written as package (keyword). an example is (package main)

2. Import packages: to import or bring in filels included in a package. example (fmt), (string).

3. Functions: any code inside the the functin main (func main() {}) will be executed.

4. Statements and expressions: is a function made available from the fmt package.used for printing an output or text. example, fmt.Println("word").
