## SENTINEL STRING TRANSFORMER — ONLINE

SENTINEL STRING TRANSFORMER is a simple command-line tool that lets you transform text in different ways.

# You can transform text with these different commands:

1. upper: Converts text to uppercase. e.g upper sentinel is watching.

2. lower:  Converts text to lowercase. e.g lower SENTINEL IS WHATCHING.

3. cap: Capitalizes the first letter of each word. e.g cap director adaeze okonkwo.

4. title: Title case, small words remain lowercase. eg. title the rise of the eastern threat.

5. snake: Converts text to snake_case. eg. snake Operation Gopher Protocol.

6. reverse: Reverses each word individually. eg. reverse Go saves the world.
