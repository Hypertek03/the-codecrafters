## Go Variables

Variables are containers or carriers for storing data values.

There are different types of variables:

1. int: this stores integers (whole numbers), such as 123 or -123.

2. float32: stores floating point numbers, with decimals, such as 19.99 or -19.99.

3. string: stores text, such as "How is Nigeria". String values are surrounded by double quotes ("").

4. bool: stores values with two states: true or false.

Declaring or Creating a variable..

 There are two ways to declare a variable:

 1. var (word) keyword. Example: var student string = "10"

 2. := sign (value). Example: x := 10
