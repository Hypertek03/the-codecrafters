## Go Comments

1. A comment is a text which is ignored when a code is executed.

2. Comments can make codes readable, understandable and self-explanatry .

3. Comments can be used to prevent code execution when testing an alternative code.

In GO there are two types of comments which are supported:

1. single-line comments: starts with two forward slashes (//) and Any text between the slashes and at the end of the line is ignored, Will not be executed.
example: // This is a comment

2. multi-line comments:  start with /* and ends with */. (forward slash and asterisk), Any text between /* and */ will be ignored and won't be executed.
example: /* this is a comment
, and it is amazing */
