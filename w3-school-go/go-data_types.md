Data type is one of the important concept in programming which specifies the size and type of variable values.

in Go there are three basic data types:

1. bool: represents a boolean value and is either true or false.
2. Numeric: represents integer types, floating point values, and complex types.
3. string: represents a string value.

# Boolean: 
Data type is declared with the "bool" keyword and can only take the values true or false.

The default value of a boolean data type is false.

# Integer: 
Data types are used to store a whole number without decimals, like 35, -50, or 1345000.

* The integer data type has two categories:

    Signed integers - can store both positive and negative values.

    Unsigned integers - can only store non-negative values.

* The defult type for integer is int.

# Float:
Data types are used to store positive and negative numbers with a decimal point, like 35.3, -2.34, or 3597.34987.

Float data type has two keywords:
1. Float32
2. Float64

# String:
Data type is used to store a sequence of characters (text). String values must be surrounded by double quotes. ("")
