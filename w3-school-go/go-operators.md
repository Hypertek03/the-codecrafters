## Operators
Are used to perform operations on variables and values.

The + operator adds together two values.
Although the + operator is often used to add together two values, it can also be used to add together a variable and a value, or a variable and another variable:

operators are divided into the following groups:

1. Arithmetic Operators: Arithmetic operators are used to perform common mathematical operations. e.g

+ Addition. Adds together two values.

- Subtraction. Subtracts one value from another.

* Multipication. Multiplies two values.

/ Dvition. Divides one value by another.

% Module. Returns the division remainder.

++ Increment. Increases the value of a variable by 1.

-- Decrement. Decreases the value of a variable by 1.


2. Assignment Operators: Are used to assign values to variables.

* Use the assignment operator (=) to assign the value 10 to a variable called x:

3. Comparison Operators: Comparison operators are used to compare two values.

* The return value of a comparison is either true (1) or false (0).

* We use the greater than operator (>) to find out if 5 is greater than 3:

A list of all comparison operators:

1. == Equal to.

2. != Not Equal.

3. > Greater than.

4. < Less than.

5. >= Greater than or equal to.

6. <= Less than or equal to.

4. Logical Operators: Are used to determine the logic between variables or values:

&&. Returns true if both statements are true.
||. Returns true if one of the statements is true.
!.  Reverse the result, returns false if the result is true.

5. Bitwise Operators: Reverse the result, returns false if the result is true:

&  	AND 	Sets each bit to 1 if both bits are 1.

| 	OR 	Sets each bit to 1 if one of two bits is 1.

 ^ 	XOR 	Sets each bit to 1 if only one of two bits is 1.

<< 	Zero fill left shift 	Shift left by pushing zeros in from the right.

>> 	Signed right shift 	Shift right by pushing copies of the leftmost bit in from the left, and let the rightmost bits fall off.
